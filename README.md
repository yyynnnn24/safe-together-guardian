# SafeTogether – Female-Friendly Safety Platform

A web-based personal safety application designed to enhance women’s security through real-time location sharing, emergency alerts, and trusted contact management. Built as a full-stack MVP focusing on usability, responsiveness, and safety-first UX design.

---

##  Project Overview

**SafeTogether** is a safety-focused platform that allows users to quickly alert trusted contacts, share live locations, and report unsafe incidents. The system is designed with a clean, mobile-friendly interface to ensure fast access during emergencies.

This project was rebuilt as a full MVP after GitHub repository loss, focusing on restoring core functionality and scalable architecture.

---

##  Key Features

###  Emergency SOS System

* One-click SOS button
* Sends emergency alert with user location
* Designed for rapid response situations

###  Live Location Sharing

* Share real-time location with trusted contacts
* Helps friends/family track user safety

###  Trusted Contacts Management

* Add, edit, and remove emergency contacts
* Stores contact list securely (local storage / backend-ready)

###  Safety Check-In Timer

* Users set a safety timer
* If no response before timeout, system triggers alert

###  Incident Reporting

* Report unsafe incidents with description and location
* Helps build awareness of risky areas

###  Map Integration (MVP / Mock-supported)

* Displays user location
* Shows reported incidents (mock or dynamic data ready)

##  Tech Stack

* **Frontend:** React (or Lovable-generated React framework)
* **Styling:** Tailwind CSS / modern responsive UI design
* **State Management:** React Hooks
* **Storage:** Local Storage / extendable to backend API
* **Optional Backend:** Node.js / Express (future expansion)
